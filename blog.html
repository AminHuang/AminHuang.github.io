1.瀑布流布局

关于瀑布流布局，上网查了一下，最简单的似乎就是直接用float了。我是分了三列，第一列宽度为50%,第二列和第三列宽度为25%，然后第一列里面大图特殊处理，之后再分两列。不过这个如果大图在中间，还要动态插入的话我就不知道该怎么办了。

2.input里面的图片与文字处理

就是search那里。我是在input里面使用了背景图片，然后padding-left设置了对应大小。

3.IE兼容问题

我在过程中使用了HTML5标签，例如header和footer，测试的时候发现IE8就不兼容。为了兼容，需要手动创建且标签。直接使用html5shiv(和html5shim是一样的东西)。

4.12

css

三列等高布局

padding和margin相抵消

我一开始用的FLEX，然而IE9也并不支持。要IE兼容的话，得用那种嵌套好几个div的方式。圣杯布局和双飞翼布局都是不能几列同步高度的。

后来在群上看到别人说的设置很大的padding和margin相互抵消就好了！父元素记得overflow:hidden

.show-warp { height: auto; overflow:hidden; }

.show { float:left; width: 305px; padding-bottom: 10000px; margin-bottom: -10000px; }
